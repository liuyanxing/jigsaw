# Cocos Creator 3.8.8 · 运行指南

## M2 已就位:三屏完整流程(世界地图 → 选关 → 对局)

代码与资源都在 `assets/`。**场景仍只需"建一次 + 放一个 Boot 节点"**(同 M1),Boot 现在驱动全流程。

### 跑起来

1. **打开/聚焦工程**:编辑器会**导入新增资源**(`resources/levels.json`、`resources/maps`、`resources/icons`、`resources/levels/*`)并编译 TS。等左下角进度条跑完。
   - 确认 `资源管理器` 里这些都出现了(图片有展开三角 → 含 `texture`/`spriteFrame` 子资源)。
2. **建场景**(若还没建):`assets/` 右键 → `创建 → Scene`,命名 `Main`,双击打开。
3. **挂 Boot**:`层级管理器` 选 `Canvas` → 右键 `创建 → 空节点`(命名 `Boot`)→ 选中它 → `属性检查器` → `添加组件 → 自定义脚本 → Boot`。
4. **保存并运行** ▶。

### 期望

- 进首页**世界地图**(法国地图背景 + 5 个地点圆标 + 名牌)。
- 点某地点 → **选关页**(标题 + 5×5 关卡卡:第 1 关可点、其余🔒;缩略图为该国场景图)。
- 选一关 → **开始拼图** → 进对局(切片打乱、拖动交换、成组、归位)。
- 归位完成 → 控制台 `[Boot] 完成 france-01!`;点**← 返回** → 回选关页,**第 1 关变 ✓、第 2 关解锁**(进度持久化,重开仍在)。

### 若有问题(blind code,把控制台/现象发我)

- **块小、缝大 / 上下颠倒** → 见下方 BoardView 注释(`Slicer.ts` 的 y、`setContentSize` 顺序)。
- **图全黑 / 缩略图不显示** → 多半是资源没导入完,或 `SpriteFrame` 子路径名不同(个别为 `/spriteFrame` vs `/texture`);`Res.ts` 里我用的是 `<path>/spriteFrame` 与 `<path>/texture`。
- **点地点/卡片没反应** → 触摸命中区是节点 `UITransform`,贴日志我调。
- **`levels.json` 加载失败** → 确认它在 `resources/` 根、导入为 `JsonAsset`。
